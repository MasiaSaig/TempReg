Draws given int number to screen, at given coordinates, using given colors.

Draws given float number, with its nth mantysa digits (where n is number provided in precision parameter), to screen using font, defined in 'asciiLib.c' written in ASCII 16x8. With apropriate colors, given by user. It uses drawLetter() function.

Parameters
[in]xX coordinate on screen.
[in]yY coordinate on screen.
[in]numberGiven float number to draw.
[in]precisionNumber of digits, after decimal to draw.
[in]fontColorColor of a letter.
[in]backgroundColorColor of a background, pixels not defined in letter struct.
Returns
